Function composition

Function.pipe
  • fn1
  • fn2
  • fn3
function
  • opal.pipe

Function.pipe composes functions as a pipeline in a “point-free” manner. Applying the pipelined function applies the leftmost function first so that Function.pipe(fn1,fn2,fn3)(x) is the same as fn3(fn2(fn1(x))).

Function.compose
  • fn1
  • fn2
  • fn3
function
  • opal.compose

Function.compose performs “point-free” function composition in the traditional order. When the composition is applied, the rightmost function is applied first so that Function.compose(fn1,fn2,fn3)(x) is the same as fn1(fn2(fn3(x))).

fn.create
  • a
  • b
  • c
object

For a constructor fn, the method fn.create abstracts the use of the new operator. At most ten arguments can be passed to the method.

var jd = Date.create(1997,8,29);
jd instanceof Date;
true
jd.toDateString();
"Mon Sep 29 1997 00:00:00 GMT+0100 (BST)"

Object.construct
  • constructor
  • a
  • b
  • c
(value object)

Object.construct wraps the new operator in a function. Arguments to the constructor can be passed at definition-time or application-time, but no more than ten arguments in total.

Object.construct(Date)('1997-8-29');
"Fri Aug 29 1997 00:00:00 GMT+0100 (BST)"

Object.construct(Date)('1997-8-29') instanceof Date;
true

function Employee (department, forename, surname) {
    this.department = department;
    this.forename = forename;
    this.surname = surname
}
var makeITstaff = Object.construct(Employee,'IT');

var john = makeITstaff('John','Smith');
john.department;
"IT"
john.surname;
"Smith"

Object.method

  • a
  • b
  • c
(object value)

Object.method(name) is a function that invokes the named method on an object. Additional arguments can be provided at suspension time but not at invocation time.

Object.method('method',a,b,c)(object) is equivalent to object.method(a,b,c,d,e,f).

Object.method('split','-')('john-smith');
["john","smith"]

Object.path
  • elements
  • separator
(object value)

Object.path returns a function that dereferences a path on an object. The path can be specified as either an array of elements or a string specifying the elements. In the latter case, an optional separator can also be passed (the default is the ‘.’ character). If any element of the path is a method then the method will be invoked during dereferencing.

Object.path(['name','last'])({name:{first:'john',last:'smith'},age:27});
"smith"

Object.path('name.last')({name:{first:'john',last:'smith'},age:27});
"smith"

Object.path('name/last','/')({name:{first:'john',last:'smith'},age:27});
"smith"

Given arguments a,b,c..., applyto creates a function that when called on a second function applies the second function to the provided arguments, so that applyto(a,b,c...)(fn) is equivalent to fn(a,b,c...).
