Opal.js

Object, Predicate and Action Library

Opal provides primitives for function composition and object manipulation; and sub-libraries for predicates, orderings and reduction functions.

Sapphire.js

A library for sets and lists

Sapphire builds on Opal to provide an implementation of sets and lists, including typed collections with implicit constructors and indexed collections.

Emerald.js

A library for event composition

Emerald implements a model of composite events that effectively allows queries over events. It also provides primitives for combining and splitting event streams.

Topaz.js

A library for observable objects

Topaz combines Sapphire sets and Emerald events to provide observable sets and observable objects that issue notifications when changed.
